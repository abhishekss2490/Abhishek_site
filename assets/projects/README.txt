Project preview media go here. Filenames must match the page:

  edgelio.jpg        (EdgeLIO — a mapping/point-cloud screenshot)
  etc-rl.jpg         (Event-Triggered Safe RL — a results plot, e.g. inter-event time vs. beta)
  sceneforge.mp4     (SceneForge — a short screen-capture of the live demo)

Images: .jpg or .png (16:10 looks best, e.g. 1280x800).
Video:  .mp4 (H.264), muted, a few seconds, loops automatically. Keep under ~5 MB.

Until a file exists, the preview shows a saffron placeholder with the project name.
To change which file a project points to, edit the data-src on its .prow in hf0-inspired.html.
