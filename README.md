# Abhishek S — Portfolio Site

Personal portfolio for MS/PhD applications (Fall 2027). Robotics · Control · Reinforcement Learning.

## The site
- **`design-previews/hf0-inspired.html`** — the current site. Single-page, **no-scroll**, tabbed
  (Research · Publications · Projects · Experience · Education & Skills · Beyond + CV). Open directly
  in a browser (loads Google Fonts + local assets).
- `design-previews/assets/` — headshot, project images, the saffron sphere spritesheet, etc.
- Older design explorations also live in `design-previews/` (monospace-terminal, swiss-editorial,
  old-web, dashboard, tabbed, etc.). `hf0-inspired.html` is the chosen one.

## Design language
HF0-inspired: **milky white `#FCFBF7` + saffron `#FC8019`**, IBM Plex Mono + Inter, a signature
**sliced-sphere** animation (recoloured to saffron from HF0's GIF frames), grain overlay, a mini
spinning-sphere logo, and a faint ghost sphere behind the inner tabs.
