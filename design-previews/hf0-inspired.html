Projects
01
EdgeLIOPES University, Bangalore
SLAM / Perception
Onboard LiDAR–inertial odometry and mapping (FAST-LIO2, Livox MID360, source-built ROS 2 Humble) running entirely on a Jetson Xavier NX — no offboard compute. Real-time state estimation and 3D reconstruction: 10 Hz mapping sustained over a 15-min indoor run at ~22% CPU, 15 ms/scan latency, and 0.13 m ATE RMSE.
GitHub
02
Event-Triggered Lyapunov-Safe Reinforcement LearningIISc, Bangalore
Research Project · Safe RL / Control
Cast event-triggered control of a linear system as a constrained MDP; learned safe, state-dependent triggering via Q-learning → DQN → SAC under a Lyapunov certificate with certified-set projection (Zeno-free). Proved the optimal inter-event time has intrinsic jump discontinuities in the trigger parameter β. Advised by Dr. P. Tallapragada.
